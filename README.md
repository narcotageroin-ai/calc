
# Калькулятор цен для маркетплейсов (v3)

Версия v3 с окончательной логикой НДС и маржинальности:

- **Исходящий НДС** считается как `(цена продажи - СПП) * ставка_НДС / (100 + ставка_НДС)`
  (для 22% это `* 22 / 122`).
- **Входящий НДС** теперь считается **по той же ставке**, то есть:
  `входящий НДС = закупка * ставка_НДС / (100 + ставка_НДС)`
  (например, `закупка * 22 / 122`).
- Маржинальность проверяется **не от цены реализации**, а от суммы,
  которая остается после оплаты комиссии МП, эквайринга и логистики.

Формула базы маржинальности:

`base = Цена продажи - Комиссия МП - Эквайринг - Логистика`

Прибыль считается после всех затрат (закупка, логистика, упаковка, комиссии, эквайринг, НДС к уплате),
и целевая маржа — это:

`Маржа = Прибыль / base`

Если фактическая маржа < минимальной (по умолчанию 10%), цена пересчитывается аналитически так,
чтобы маржа была ровно на уровне порога (и при этом цена не становится ниже исходной, полученной по наценке).
